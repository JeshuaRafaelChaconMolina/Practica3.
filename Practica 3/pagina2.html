DISEÑO RESPONSIVO
Elementos clave de un diseño responsive
  • Tamaño de navegación.
  • Formato de visualización.
  • Contenidos en texto.
  • Contenidos visuales.
  • Marca o logo.
  • Llamadas a la acción.
Ventajas de tener un sitio web responsive
  1. Es importante para el SEO.
  2. Mejora el branding corporativo.
  3. Se reduce en gran medida el tiempo de desarrollo de la web.
  4. Proporciona una mayor viralidad de los contenidos y las imágenes.
  5. El tiempo de carga se reduce exponencialmente.
  6. Se evitan los contenidos duplicados.
